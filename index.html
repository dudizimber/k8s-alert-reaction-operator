Alert Reaction Operator Helm Repository

This is the official Helm repository for the Alert Reaction Operator.

Add Repository

helm repo add alert-reaction-operator https://dudizimber.github.io/k8s-alert-reaction-operator/
helm repo update

Install Chart

helm install my-operator alert-reaction-operator/alert-reaction-operator

Available Charts

  • index.yaml - Helm repository index
  • alert-reaction-operator-0.1.0.tgz
